PROJECT OVERVIEW

INTRO

The Flexible Work Location Tool prototype was developed by 2 people as a part of the capstone project for 'Okanagan College'

PROBLEM

Scheduling has been a key aspect of everyday life for centuries. It is especially important as a student to keep track of what needs to be done for each day.

GOAL

The Flexible Work Location Tool was developed as a solution to aid with complying with provincial work location policy as well as reduce the time spent by employees creating, managing, filling and reviewing forms. The user interface is designed to be scalable and flexible.

USERS

There are four types of users within the application
